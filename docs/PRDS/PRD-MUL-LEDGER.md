# FIRELight 多账本管理 (Multi-Ledger) PRD

## 1. 账本管理模块

**功能描述**：支持用户创建和管理多个独立账本，满足不同场景的记账需求（如日常消费、投资理财、家庭共同账本等）。

**主要功能点**：
- 创建账本：设置账本名称、图标/颜色标识
- 编辑账本：修改账本基本信息
- 删除账本：删除账本及其所有条目
- 账本切换：快速切换当前操作的账本
- 账本概览：显示账本余额、本月收支统计

**数据模型**：
| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 账本唯一标识 |
| name | String | 账本名称 |
| icon | String | 图标标识 |
| color | String | 主题色 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |
