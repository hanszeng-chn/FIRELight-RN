# FIRELight 表格导入 (Table Import) PRD

## 表格导入模块

**功能描述**：支持从外部表格（CSV/Excel）批量导入历史记账数据，降低用户迁移成本。

**主要功能点**：
- 文件上传：支持 CSV、XLSX 格式
- 字段映射：自动识别或手动映射表格列与系统字段
- 数据预览：导入前预览解析结果，支持修正
- 冲突检测：检测重复条目,支付后退款等特殊情况，支持跳过/覆盖策略
- 导入历史：记录每次导入操作，支持回滚

**导入流程**：
```
上传文件 → 解析表格 → 字段映射 → 数据预览 → 确认导入 → 完成
```

**支持的映射字段**：
- 日期 (必填)
- 金额 (必填)
- 类型 (收入/支出)
- 分类
- 备注/描述
